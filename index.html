About Me

Saya adalah seorang mahasiswa yang bersemangat dalam pengembangan web, dengan fokus sebagai full stack web developer. Saya percaya bahwa teknologi web memiliki potensi besar untuk mengubah cara kita berinteraksi dengan dunia, dan saya berkomitmen untuk terus belajar dan mengembangkan keterampilan saya dalam merancang dan membangun pengalaman web yang menarik dan responsif. Dengan latar belakang pendidikan di bidang Teknik Informatika dan hasrat saya dalam coding, saya selalu mencari peluang untuk menciptakan solusi inovatif yang dapat memberikan dampak positif pada dunia digital. Saya juga senang berkolaborasi dengan orang-orang kreatif lainnya dalam pengembangan proyek-proyek menarik. Sebagai seorang pelajar full stack web developer saya memiliki pengalaman dalam merancang dan mengembangkan User Interface yang responsif dan menarik menggunakan HTML, CSS, Javascript.
